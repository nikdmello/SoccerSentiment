## Overview
A sentiment analysis iOS app made specially for Premier League soccer fans who want to gauge the overall sentiment, based on the latest Tweets, of any 2020 Premier League team.

## Libraries used
- SwifteriOS
- SwiftyJSON

## Frameworks used
- Core ML
- Create ML
